¿Qué es EAN-8?

EAN-8 es un código de barras de 8 dígitos utilizado para productos pequeños donde no hay espacio suficiente para un código EAN-13 completo. Los primeros 7 dígitos identifican el producto y el último es un dígito de control calculado automáticamente.
